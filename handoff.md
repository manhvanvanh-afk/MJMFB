### 轮次对照表

| round 字段 | 标签 | 日期范围 | 场次 |
|-----------|------|---------|------|
| `group` | 小组赛 | 06-12 ~ 06-27 | 72 |
| `r32` | 1/16决赛 | 06-28 ~ 07-03 | 16 |

| `thirdPlace` | 季军赛 | 07-19 | 1 |

### 投注类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `胜平负` | 猜一场比赛胜/平/负 | 墨西哥 vs 韩国 猜墨西哥胜 |
| `总进球` | 猜一场比赛总进球数 | 2球 / 3球 |
| `半全场` | 猜半场+全场结果 | 胜胜 / 平负 |
| `比分` | 猜精确比分 | 2:1 |
| `串关` | 多场比赛串联 | 8串1 / 10串1 |

## 四、每日运营流程

### 核心原则

> **你只管告诉我押注和赔付情况，Excel 和网站数据我来同步维护。**

### 你的工作

每天把群里的押注情况发给我，说清楚就行。例如：

```
今天押注情况：
- 佑仔 串关8串1 200分 赔率35倍 中了
- 凡强 胜平负 墨西哥vs韩国 100分 赔率2.0 中了
- 阿全 比分 巴西vs摩洛哥 猜3:1 50分 赔率8倍 没中
```

### 我的工作

收到押注数据后，我会：

1. **更新 Excel** → 自动填入「押注明细」Sheet
2. **自动爬比分** → `python3 scrape_eblcu.py`（从 eblcu.net 获取）
3. **更新 data.json** → 排行榜/投注记录
4. **毒奶榜 / 锐评 / MVP** → 按需更新
5. **推送到 GitHub**

## 五、Excel 记账表说明

文件路径：`/Users/guolong/Documents/MJMFB/世界杯竞猜记账表.xlsx`

### 三个 Sheet

| Sheet | 作用 |
|-------|------|
| **押注明细** | 每一笔押注的详细记录（核心表） |
| **每日汇总** | 每天的总账（手动填写） |
| **个人统计** | 每个人的总体战绩（手动填写） |

### 列说明

| 列 | 说明 |
|----|------|
| 日期 | 押注日期 |
| 姓名 | 下拉选择（19位同学） |
| 投注类型 | 下拉选择：胜平负/总进球/半全场/比分/串关 |
| 比赛详情 | 写清比赛+押注内容 |
| 押注金额(分) | 用户投了多少分 |
| 赔率 | 中了之后赔多少倍 |
| 赔付金额(分) | 自动计算：=押注金额×赔率（仅中奖时） |
| 净赚(分) | 自动计算：=赔付金额-押注金额 |
| 中奖 | 下拉选择：✅ 中 / ❌ 没中 |
| 备注 | 备注信息 |

> 这个 Excel 由 Codex（我）维护更新，你只需要每天告诉我押注情况即可。

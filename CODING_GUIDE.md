# 2026 世界杯老同学联赛 — 开发规范

---

# 一、项目目标

本项目是一个：

# "世界杯娱乐战报静态网站"

核心目标：

- 适合 GitHub Pages 部署
- 易维护
- 易手动更新
- 娱乐感强
- 体育氛围强
- 小白开发友好

---

# 二、技术限制（非常重要）

## 必须遵守：

### 使用：

- HTML
- CSS
- Vanilla JavaScript
- JSON 数据文件

---

## 禁止使用：

- React
- Vue
- Angular
- Node.js 服务端
- 数据库
- 登录系统
- 后端 API
- TypeScript
- SSR
- Webpack
- Vite
- npm 复杂工程

---

# 2026 世界杯老同学联赛 — 产品需求文档 (PRD)

> **项目代号**：WorldCup League 2026  
> **版本**：v1.0  
> **更新日期**：2026-05-19  
> **状态**：待开发

---

## 一、项目背景

### 1.1 为什么做这个项目

老同学微信群（约19人）在2026世界杯期间需要一个娱乐竞猜互动平台。

### 1.2 核心目标

1. 增强微信群互动氛围
2. 提供世界杯期间的"战报"展示
3. 形成世界杯期间的共同回忆
4. 用极低技术成本实现长期运营

### 1.3 项目定位

**"世界杯娱乐战报网站"**

不是：
- ❌ 真正博彩网站
- ❌ 实时赔率系统
- ❌ 商业产品

所以：
- ✅ 所有数据允许手动录入
- ✅ 不需要登录系统
- ✅ 不需要数据库
- ✅ 不需要后端
- ✅ 不需要支付
- ✅ 不需要自动抓赔率

**网站核心 = 娱乐氛围 + 排行榜 + 战报**

---

## 二、技术方案

### 2.1 技术目标

最简单、最稳定、最容易维护。

### 2.2 技术栈

| 技术 | 用途 |
|------|------|
| HTML | 页面结构 |
| CSS | 样式与动画 |
| Vanilla JavaScript | 交互逻辑 |
| JSON | 数据存储 |
| GitHub Pages | 免费部署 |

### 2.3 明确排除

不使用：
- React / Vue / 任何前端框架
- Node.js
- 数据库
- API 服务
- 构建工具

### 2.4 部署方案

通过 GitHub Pages 部署，推送即更新。

参考：[GitHub Pages 官方文档](https://pages.github.com/)

## 四、首页布局（核心页面）

### 4.1 顶部 Hero 区

**内容**：
- 巨大的世界杯标题
- "WorldCup League 2026"
- 老同学世界杯联赛
- 世界杯风格背景图
- 足球、球场、灯光等视觉元素

**风格参考**：FIFA 官网、ESPN、体育酒吧大屏幕

**视觉要求**：
- 深色背景
- 足球草坪纹理
- 发光文字
- 动态灯光
- 金属感卡片

### 4.2 今日赛果模块（Today's Matches）

显示当日比赛结果，例如：

```
阿根廷  2 : 1  法国
葡萄牙  3 : 0  墨西哥
```

**数据来源**：手动更新 JSON

### 4.3 今日赢家模块（Daily Winners）

显示当日积分变化前三名：

```
🥇 张三  +12
🥈 李四  +8
🥉 王五  +5
```

**作用**：增强每日竞争感

### 4.4 总排行榜模块（Leaderboard）

显示所有成员总积分：

| 排名 | 用户 | 总积分 |
|------|------|--------|
| 1 | 张三 | 128 |
| 2 | 李四 | 120 |

**UI 要求**：前三名使用 🥇🥈🥉 特殊卡片样式

### 4.5 毒奶榜模块（Curse Ranking）

娱乐模块，展示"反向预测"战绩：

```
- 李四连续3次看衰阿根廷失败
- 王五连续预测英格兰翻车
```

**作用**：增强娱乐性

### 4.6 AI 锐评模块（重点模块）

搞笑毒舌的体育解说风格评论：

```
"系统检测到王总已进入情绪投注阶段。"
"张三已经连续三天相信英格兰。"
"李四目前处于世界杯幻觉状态。"
```

**风格**：搞笑、毒舌、体育解说风

### 4.7 今日 MVP 模块（今日球神）

展示当日最佳预测者：

```
"张三精准命中 2:1 比分预测。"
```

## 六、UI / UX 设计要求

### 6.1 风格关键词

世界杯 · 体育酒吧 · 足球激情 · 深夜观赛 · 体育直播感

| 角色 | 颜色 |
|------|------|

| 强调色 | 草坪绿 |
| 高亮色 | 金色 |
| 辅助色 | 荧光蓝 |

### 6.3 动效要求

**允许**：
- 发光动画
- 卡片 hover 效果
- 滚动动画
- 数字跳动效果

**禁止**：
- 复杂 SPA 交互
- 重型前端框架
- 复杂交互动效

## 八、每日运营流程

```
Step 1 ─▶ 微信群内进行预测
Step 2 ─▶ 比赛结束，管理员整理结果
Step 3 ─▶ 手动更新 data.json
Step 4 ─▶ 推送 GitHub
Step 5 ─▶ GitHub Pages 自动更新网站
```

## 九、开发原则

### 9.1 三大原则

1. **简单 > 炫技**
2. **娱乐感 > 专业感**
3. **运营感 > 自动化**

### 9.2 网站最重要的目标

> **不是技术多复杂，而是让微信群每天愿意打开。**
